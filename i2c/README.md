# i2c
RoadNarrows Robotics I<sup>2</sup>C Package

## Description
The I2C package provides basic interfaces to devices connected
to an I2C Bus.
The I2C package provides:
* The libi2c library.
* A set of commmands to interface with an I2C Bus at the byte level.
