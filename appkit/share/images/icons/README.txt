These are the standard RoadNarrows icons available for common use across 
RoadNarrows developed packages and applications.

Color schemes are chose from www.colorcombos.com color schemes.

For the current set of icons, ColorCombo430 is used
  (http://www.colorcombos.com/color-schemes/430/ColorCombo430.html)

Other color schemes and/or look and feel motifs may be used. I recomened that
the developer creates a subdirectory under this directory to place all of the
icons.

All icons can include black #000000 and/or white #ffffff.
